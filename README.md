## Dishine E-Commerce API

### About

Dishine E-Commerce API is a RESTful API system designed to power online store operations, manage products, process orders, and handle user accounts efficiently. Built with scalability and modularity in mind, this API allows seamless integration for various e-commerce needs, from product management to customer experience enhancements.

This project was developed as a school assignment.

### Tech Stack

This project is built using the following technologies:

- **Backend:** [Laravel](https://laravel.com/) (PHP)
- **Frontend:** [React](https://react.dev/)
- **Database:** MySQL
- **Design & Prototyping:** [Figma](https://www.figma.com/)

- **Authentication:** Laravel Sanctum
